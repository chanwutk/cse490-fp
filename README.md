# Convolutional Neural Network Visualizer

## To start the demo

1. Start the server

```
cd server
pip3 install -r requirements.txt
python3 app.py
```

2. Start the gui

using `yarn`:
```
cd gui
yarn
yarn build
yarn serve -s build
```
